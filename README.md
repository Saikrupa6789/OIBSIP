# Todo List - HTML, CSS, and JavaScript

Welcome to the Todo List project! This simple web application allows you to create, delete, and mark tasks as complete. The interface neatly segregates pending and completed tasks, providing an organized and user-friendly experience.

## Getting Started

To run the Todo List locally, follow these steps:

1. **Clone the Repository:**
   ```
   git clone [https://github.com/Saikrupa6789/OIBSIP.git]
   ```

2. **Open `main.html`:**
   Open the `main.html` file in your preferred web browser.

3. **Manage Your Tasks:**
   - Create new tasks using the input field.
   - Delete tasks by clicking the delete icon.
   - Mark tasks as complete by clicking the checkbox.
   - View pending and completed tasks separately for better organization.

## Project Structure

The project follows a simple structure:

- **`main.html`:** Contains the HTML structure for the todo list interface.
  
- **`styles.css`:** Includes the CSS styles for layout and aesthetics.

- **`script.js`:** Implements the todo list logic using JavaScript.

## Features

- **Add Tasks:**
  - Enter a task in the input field and press enter to add it to the list.

- **Delete Tasks:**
  - Click the delete icon next to a task to remove it from the list.

- **Mark as Complete:**
  - Check the checkbox to mark a task as complete.

- **Separate Views:**
  - Pending tasks are displayed separately from completed tasks for easy tracking.

## Usage

1. **Add a Task:**
   - Enter a task in the input field.
   - Press enter to add the task to the list.

2. **Delete a Task:**
   - Click the delete icon next to a task to remove it.

3. **Mark as Complete:**
   - Check the checkbox next to a task to mark it as complete.

4. **View Separate Lists:**
   - Pending tasks and completed tasks are displayed in separate sections.

## Customization

Feel free to customize the styles and layout in the `styles.css` file to match your preferences. You can also extend the functionality by adding features such as task prioritization, due dates, or categorization.
